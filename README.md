# 🎁 送礼灵感生成器

基于 AI 的个性化礼物推荐工具，使用 Ant Design 6.1.4 构建，支持手机端和电脑端响应式设计。

## ✨ 功能特性

- **基础信息输入**：支持输入收礼人的性别、年龄、兴趣爱好（可选）
- **预算设置**：可设定预算范围（最低和最高预算）
- **AI 礼物推荐**：基于输入信息，调用 LLM API 生成 3 个个性化礼物选项
- **重新生成**：支持点击"换一批"获取新的推荐
- **API 配置**：提供 UI 界面配置 API Key、API URL 和 Model
- **响应式设计**：完美适配手机端和电脑端

## 🚀 快速开始



## 📋 使用说明

### 1. 配置 API

首次使用前，需要配置 LLM API：

1. 点击页面顶部的"API配置"折叠面板
2. 填写以下信息：
   - **API URL**：默认为 `https://api.siliconflow.cn/v1/chat/completions`
   - **API Key**：输入你的 API Token
   - **Model**：默认为 `Qwen/Qwen3-8B`
3. 点击"保存配置"按钮

配置会自动保存到浏览器本地存储，下次打开无需重新配置。

### 2. 填写收礼人信息

- **性别**：选择收礼人的性别（必填）
- **年龄**：输入收礼人的年龄（必填）
- **兴趣爱好**：可选，例如：阅读、咖啡、摄影、运动等

### 3. 设置预算

- **最低预算**：输入最低预算金额（元）
- **最高预算**：输入最高预算金额（元），必须大于最低预算

### 4. 生成推荐

点击"生成礼物推荐"按钮，系统将基于输入信息调用 LLM API 生成 3 个礼物推荐。

### 5. 重新生成

如果对推荐结果不满意，可以点击"换一批"按钮重新生成新的推荐。

## 📝 API 说明

应用使用 OpenAI 兼容的 API 接口，支持以下服务：

- SiliconFlow（默认）
- 其他兼容 OpenAI API 格式的服务

### API 返回格式

应用会解析 API 返回的 `choices[0].message.content` 字段，并提取格式化的礼物推荐：

```
1. [礼物名称] - [特点描述]
2. [礼物名称] - [特点描述]
3. [礼物名称] - [特点描述]
```

## 🔒 隐私说明

- API Key 仅存储在浏览器本地存储中，不会上传到任何服务器
- 所有数据仅在本地处理，不会收集用户信息
